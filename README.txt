Measure temperature with ds18b20 and arduino.
Program show temperature in serial monitor.
If you want to change frequency between reading next value, 
type letter(only 1 character) to serial port:
a - 5 sec between next measure
b - 60sec
c - 5minutes
d - 10min
e - 15min
f - 30min
g - 1hour

Program allow communicate with my program Temperature writen in C#. 
Its GUI with aquisition to .xls file.
